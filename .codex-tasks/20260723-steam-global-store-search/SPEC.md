# Steam 全地区商店搜索

## Goal

让 Monica Steam 的商店搜索同时查询账号地区与若干代表性地区目录。账号地区锁区的游戏仍可出现在搜索结果中，并在列表与详情页明确显示当前账号地区的售卖状态。详情页继续提供多区价格查看入口。

## Scope

1. 搜索结果合并账号地区和代表性地区目录，并按名称相关度去重排序。
2. 使用账号地区的商店元数据确认候选商品是否可购买。
3. 账号地区详情缺失时，使用发现该商品的地区加载公开详情。
4. 锁区商品禁用本地购物车与愿望单购买操作，保留官方页面和多区价格入口。
5. 增加中英文文案、单元测试和架构守卫。

## Constraints

1. 账号货币与当前地区价格仍以 Steam 账号地区为准。
2. 其他地区价格必须明确标注为参考地区价格。
3. 保持现有扫码、账号会话、购物车、愿望单和多区价格实现。
4. 只运行单元测试、静态检查和 Kotlin 编译，禁止构建 APK。
5. 完成后单独提交并推送 `main`。

## Acceptance Criteria

1. 账号地区搜索结果与其他地区搜索结果能够去重合并。
2. 锁区结果显示“当前账号地区不售卖”。
3. 锁区结果可进入详情页，详情页可打开多区价格。
4. 锁区详情无法加入本地购物车或修改愿望单。
5. 商店相关测试、全量单元测试与 `compileDebugKotlin` 通过。
